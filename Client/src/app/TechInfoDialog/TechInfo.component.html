Client

Client side is an Angular 8 / TypeScript application running in a Windows 10 environment as an Application Service on Azure. Client was written exclusively to test/show the Server side application. The pointer to the server is contained in the file Configuration.ts. An artifical delay is put into the server accesses in file app.component.ts on approximately line 105.

Server

Server is an ASP.NET Core 2.2 application running as an Application Service on Azure. Server uses Entity Framework as the ORM to access the Azure based SQL database. Changing the runningOnAzure flag in appsettings.json is the sole change that has to be made in order to run locally or on Azure.

Server uses Cross-Origin Resource Sharing (CORS) to limit access to the server to only the client application. Database user name and password are stored in an Azure Key Vault to prevent being visible to others. Database injection attacks are deterred by using the Entity Frameworks Linq interfaces.

Server consists of approximately 500 C# statements (semicolon count) and 224 lines of comments (1800 lines overall). Data in the SQL database is seeded automatically with three users ('Richard Bousman', 'Trent Wignall' and 'Fred Flinestone'). Database migrations are set up automatically to allow modifications in the future. A message handler is implemented which will show the error message, HTTP request being processed (if applicable), exception information (if applicable) and source file, method and line number that generated the message. Currently all messages are ultimately routed to the console.

Testing

Unit testing for the server consists of automated tests in which 87 different assertions are run in 13 distinct areas: Database seeding, Address handling, Person creation, People Get, People Post, People Put, People Delete, Creating Interests, Adding and deleting interests separatly, Adding and deleting interests in the same call, Creating Images, Keeping track of number of users using an image and Image controller. Unit test code makes up nearly 1100 lines of C# code with 215 of those lines being comments.
